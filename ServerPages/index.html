Leloutama

An web-server written entirely in PHP, along with multi-threading.

You successfully started the web-server, and this is the default page of the server.

Where is this page's source located?

This html page can be found in here: /path/to/Leloutama/src/lib/Ext/ServerPages/index.html, and if you wish to edit it you can but it is rather recommended to create another file in the DocumentRoot directory, and create a router to point to it, and start the server.

Which router does it use?

C'mon you should know the router, because you must have started the cli/run.php with those arguments, didn't you? Well, you know, so the router is most probably /path/to/Leloutama/examples/router.example.php unless you've done something clever ;-)

What's next?

Create a router, invoke shit@shitty-pc /path/to/Leloutama $ php cli/run.php --host <HOST_NAME> --port <YOUR_PORT> --router </path/to/router.php> , and... PROFIT!
