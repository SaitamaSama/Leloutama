Leloutama

Leloutama is a multi threaded web server app written in PHP.

Latest Release

Get started

$ git clone https://github.com/SaitamaSama/Leloutama && cd Leloutama
$ cli/install
...
$ cli/run --router examples/router.example.php
...

Features

  • Supports middlewares in the form of extensions
  • Supports gzip and deflate output compression
  • Supports HTTP Caching
  • Supports friendly urls
  • Supports twig templating engine
  • No additional softwares except PHP
  • Beautifully designed codebase ... ?

Drawbacks

  • Is not stable
  • No SSL/TLS support
  • No support for websocket till now
  • Un-Documented codebase

! Important !

Currently there is an problem that the server seg faults after running for a while.
